### 总体设计

- **本项目用到的技术和框架**<br>

|  工具 | 名称 
| ------------ | ------------
| 开发工具  | IDEA、vscode
| 项目构建 | Maven
| UI框架 | layui、semantic UI
|  语言 | JDK1.8、HTML、css、js 
| 内容展示 | wangEditor.md
| 数据库  | Mysql5.6 
| 项目框架  | SSH
| ORM  | JPA 
| 安全框架  | JWT
| 运行环境  | 腾讯云Centos7
